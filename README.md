# COMP 345 — Assignments

Course assignments for **COMP 345** (Computational Linguistics and Natural Language Processing), implemented in Jupyter notebooks.

---

## Contents

| Assignment | Description |
|------------|-------------|
| **A1** | `NehirOzs_A1.ipynb` — Assignment 1 |
| **A2** | `NehirOzs_A2.ipynb` — Classification metrics, linear regression from scratch, Ridge/Lasso regularization, and logistic regression for text classification using a 7-book dataset and bag-of-words features |

---

## Assignment 2 Overview

Assignment 2 covers:

- **Classification evaluation** — Confusion matrix, precision, recall, F1-score
- **Linear regression** — Implementation from scratch
- **Regularization** — Ridge and Lasso
- **Text classification** — Logistic regression with bag-of-words on the 7-book dataset

---

## Setup & Running

1. **Environment**: Python 3 with Jupyter (or open in Google Colab).
2. **Dependencies**: `numpy`, `pandas`, `scikit-learn` (and any others listed in the first code cell of each notebook).
3. **Run**: Open the `.ipynb` file in Jupyter Lab/Notebook or upload to [Google Colab](https://colab.research.google.com/), then run the cells in order.
